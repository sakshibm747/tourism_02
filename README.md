# 🌍 Tourism Travel Booking System

A web-based tourism booking platform that allows users to explore travel destinations, view curated tour packages, and book trips easily. The system connects travelers with various travel packages and provides a simple interface for browsing destinations and planning trips.

---

# 🚀 Features

* 🌄 Explore tourism packages and destinations
* 📅 View detailed travel itineraries
* 🧾 Simple booking interface
* 🔐 Google Authentication for secure login
* ☁️ Cloud database integration
* 📱 Responsive and user-friendly design

---

# ⚙️ Installation

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/yourusername/tourism-booking-system.git
```

### 2️⃣ Navigate to Project Folder

```bash
cd tourism-booking-system
```

### 3️⃣ Install Dependencies

```bash
pip install -r requirements.txt
```

### 4️⃣ Run the Application

```bash
python app.py
```

The application will start on:

```
http://127.0.0.1:5000
```
